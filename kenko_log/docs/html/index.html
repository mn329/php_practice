PHPアプリ 未完成箇所調査レポート課題

課題の目的

「Kenko Log」の未完成箇所や動作不良を自分で調査し、原因と修正方針をレポートとしてまとめる。

  • エラーや表示崩れを見つける
  • PHP、JavaScript、SQL、HTML の調査・役割を説明する
  • データベース定義や既存コードを根拠に原因を考える
  • 修正後に何を確認すればよいかを整理する
  • マークダウン（md）の記述訓練

提出方法・期限

提出期限

2026/07/19（日）

提出物

  • Teams にレポートを .md ファイルで提出する。
  • ファイル名: php_出席番号_氏名.md
  • 例: php_01_HAL太郎.md

必須条件

  • 下記の「未完成・要修正箇所」（配布した docs/未完成箇所レポート.md） にある項目から、最低 8項目を調査する。
  • 「未完成・要修正箇所」一覧を参考に、症状・原因・修正内容・動作確認 をまとめる。

レポート項目

観点 内容
症状の確認 実際の画面、エラー、表示内容を説明する。
原因の説明 ファイル名、変数名、SQL、API URL などを根拠に説明する。
修正方針 既存の設計や DB 定義に合った修正方針を書く。
動作確認 修正後に確認すべき操作と期待結果を書く。

記述例

php_00_HAL.mdを参考に、各項目について、次の形式でまとめる。 

### No.5 健康記録の追加

#### 症状
健康記録を追加すると SQL エラーが表示される。

#### 確認したファイル
health/insert.php

#### 原因
SQL がコーディングされていない。

#### 修正内容
SQL を以下のように修正した。

INSERT INTO ....

#### 動作確認
健康記録を追加し、一覧画面に登録内容が表示されることを確認した。

未完成・要修正箇所

No. ファイル 箇所 現象・問題
1 env.php DB 接続と作成 DB 接続情報を設定し、DB を構築する必要がある。
2 index.php ヒーローで「キャッチコピー」が表示 トップページ左側に components/top/hero_left.php を表示する。
3 components/footer.php クレジットで「2022」部分を今年で表示 フッターのクレジットの年数が「2020 - 2022」固定になっている。
4 health/index.php 心拍数(bpm)表示 健康記録の一覧表で心拍数が表示されていない。
5 health/insert.php 健康管理の記録の追加 健康記録の追加時に、存在しないテーブル名 xxxx へ INSERT しようとして SQL エラーになる。
6 health/edit.php 健康管理の編集画面表示 編集対象の id を GET パラメータから取得していないため、編集画面で「該当する記録が見つかりません。」になる。
7 api/health/csv/index.php 健康管理のCSVダウンロード CSV ダウンロード用の SQL が空のため、健康記録 CSV のダウンロード処理が失敗する。
8 activity/index.php アクティビティ種別の表示 アクティビティ一覧の「種類」列で空の配列キーを参照しているため Warning が出る。
9 activity/add.php アクティビティの記録追加 フォーム送信メソッドの不具合で「不正なリクエストです。」になる。
10 meal/index.php 食事種別表示 一覧の種類列が TODO: 食事の種別を表示 のままで、実データが表示されない。
11 sleep/delete.php 睡眠データ削除 睡眠の編集画面でデータ削除すると、Fatal error になる。
12 logout/index.php ログアウト処理 メニューからログアウトしても、完全にログアウトできない。
13 components/nav.php 認証状態による出し分け ログイン済み用ナビは $auth_user がある場合だけ表示されるようにする。未ログイン用ナビは常に include されるため、メニューが表示される。
14 login/index.php 古い入力値の復元 ログイン失敗時のセッションを復元していないため、メールアドレスが復元されない。
15 env.php Gemini API キー Google AI Studio で Gemini API キーを発行して、健康管理の AI 分析の動作確認を行う。レポートに自分の API キーは絶対に書かないこと。 https://aistudio.google.com/
16 register/input.php CSRF トークン ユーザ登録すると「不正なリクエストです。」というエラーになる。
17 register/store.php パスワードをハッシュ化して保存する ユーザ登録で password_hash がないため SQL エラーになる。
18 js/activity_chart.js const url = '' アクティビティグラフ用 API の URL が空のため、グラフデータを取得できない。
19 その他 --- ---

加点条件

次の内容がある場合は、加点対象。

  • 8項目以上の調査
  • 項目以外のバグ修正レポート
  • 現状のプロジェクトの問題点
  • さらなる改善点の提示

例: どの処理でエラーになるか、どの処理が不足していてセキュリティや保守性に影響するか、 どのような方法を利用すると改善できるかを説明する。

const GEMINI_API_KEY = 'APIキーを設定';
